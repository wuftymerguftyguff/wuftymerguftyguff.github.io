Sometimes after a ST plugin gets imported you may start getting lots of BRAIN 299 error messages in SM21, Issues accessing BI  CPH data in ST03 and BI content in E2E diagnostics. (Error loading template 0TPL_0SMD_MPEH_HA_Q0010 etc)
This can be because there are discrepancies between the active and delivered version of some of the BI content.
I have found that running the program CCMSBI_SETUP_E2E and the transaction RSTCC_INST_BIAC in the system hosting the BI for your solman will regenerate the content and fix may of these ills.
However, YMMV, so let me know how you get on.
